## What I learned this week
[Key concepts — focus on understanding, not syntax]
- HTML: Gives Page Structure.
- CSS: Gives Page Style.
- JavaScript makes it DO THINGS:
  - Respond to clicks/inputs
  - Change content dynamically
  - Fetch data from servers
  - Build Interactive apps

- document.querySelector() - Selects the first element that matches a CSS selector.
- document.querySelectorAll() - Selects all elements that match a CSS selector, returns a NodeList.
- addEventListener() - Attaches an event handler to an element for a specific event (e.g., 'click', 'input').
- **JavaScript**
  - DOM Manipulation: Changing the structure, style, or content of a webpage using JavaScript.
    - QuerySelector: Selects elements from the DOM using CSS selectors.
    - Event Listeners: Responding to user interactions like clicks or input changes.
    - Events: Actions that occur in the browser (e.g., click, input, submit) that can trigger JavaScript functions.
  - Variables: Containers for storing data values (e.g., let, const).
    - let is for values that change
    - const is for values that stay the same 
  - Data Types:
    - String: Text enclosed in quotes (e.g., "Hello").
    - Number: Numeric values (e.g., 42, 3.14).
    - Boolean: true or false values.
