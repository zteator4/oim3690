- The DOM: Think of it as a "Map" of your HTML. JavaScript uses this map to find and change things.

- The 3-Step Pattern:
  - Select: Find the element (document.getElementById).
  - Listen: Wait for a user action (addEventListener like a 'click').
  - Change: Update the page (textContent or innerHTML).
  - Events: These are triggers like click, submit, or keydown.
