## How Websites Work
- Websites display, Web apps let you interact
- Important to make the site Responsive and platform adaptive
- Viewing page source code:
-   - right click -> view page source
-  https:// - Protocol (secure HTTP)
-  www.babson.edu - Domain name (where to go)
-  /academics/index.html - Path (which page)
-  Domain = where, Path = what
 
 ## Web Request
- DNS Lookup - Find the server's IP address
- HTTP Request - Your browser asks for the page
- Server Response - The server sends back files
  - 200 - OK (success)
  - 404 - Not Found
  - 500 - Server Error
- Browser Rendering - Your browser displays the page
