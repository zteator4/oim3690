## Class Notes
- DNS stands for Domain Name System, houses the records for domain names and translates them to IP addresses
  - IP Address is a unique string of numbers that identifies each computer to communicate over a network
- Error 404 means the server could not find the requested resource
- Https cat and https dog to learn about https
- **GET**: Retrieve data, its like searching for something
  - when the url auto changes to include parameters,or bring you to a specific page, its a GET request
- **POST**: Send data to server, like submitting a form
  - When submitting a form, data is posted, not cached
- Your computer's network info:
    - Command Prompt -> ipconfig
-  DNS Lookup:
   - command prompt -> nslookup "babson.edu"
 - Ping a server:
   - command prompt -> ping "babson.edu"
- Trace the route to a server:
   - command prompt -> tracert "babson.edu"
- **What these^ commands show:**
   - ipconfig: your computers network settings
   - nslookup: look up a domains IP
   - ping: test if a server is reachable
   - tracert: show the path to a server
     - ^These exist for Debugging
